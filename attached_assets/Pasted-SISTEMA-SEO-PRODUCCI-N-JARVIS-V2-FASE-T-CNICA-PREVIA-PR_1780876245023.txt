Objetivo:

Convertir el proyecto en una aplicación lista para producción antes de realizar cualquier optimización SEO.

IMPORTANTE:

No modificar diseño.

Únicamente preparar el proyecto para despliegue profesional.

STACK REQUERIDO

• React 19
• Vite
• Tailwind CSS 3.4.17
• GSAP 3
• ScrollTrigger
• Lucide React

IMPLEMENTACIÓN OBLIGATORIA

Crear automáticamente:

1. vercel.json

Ubicación:
Raíz del proyecto

Configuración obligatoria:

{
"framework": "vite",
"outputDirectory": "dist/public",
"rewrites": [
{
"source": "/(.*)",
"destination": "/index.html"
}
]
}

No esperar a que falle el deploy.
Crear el archivo desde el principio.

2. Verificar routing SPA

Todas las rutas deben funcionar:

/
/about
/services
/contact
/pet-taxi

Al refrescar cualquier ruta:

No deben existir errores 404.

3. Configuración de build

Verificar:

• npm run build
• npm run dev
• npm run preview

Sin errores.

4. Estructura de producción

Verificar:

• Assets optimizados
• Imports limpios
• Componentes reutilizables
• Sin dependencias rotas
• Sin warnings críticos

5. Optimización inicial

Implementar:

• Lazy loading
• Dynamic imports cuando sea posible
• Component splitting
• Eliminación de código muerto
• Optimización de imágenes

6. Accesibilidad técnica

Implementar:

• aria-labels
• keyboard navigation
• semantic HTML
• focus states

Resultado esperado:

Production Ready Score:
100%
